# UI Component Library

A modern React component library built with Radix UI primitives, TypeScript, and CSS variables for consistent theming and accessibility.

## Architecture

- **Radix UI**: Headless primitives providing accessibility and keyboard navigation
- **TypeScript**: Full type safety with strict compiler options  
- **CSS Variables**: Theme-aware styling with light/dark mode support
- **Modular Scale**: Typography system using Major Third (1.25) scaling
- **Logical Properties**: International layout support with CSS logical properties

## Components

- **Props**: `variant`, `icon`, `href`, `as`, `to`, `disabled`
- **Variants**: `primary`, `secondary`
- **Usage**: Regular button, anchor link with `href`, or custom component with `as` and `to` props
- **Story**: [View Button Examples →](?path=/story/components-button--primary)

### Toggle

- **Props**: `variant`, `pressed`, `disabled`

- **Story**: [View Toggle Examples →](?path=/story/components-toggle--default)

## Theming

The library uses CSS variables for consistent theming:

```css
/* Light/Dark mode support */
[data-theme='dark'] { /* dark theme variables */ }
[data-theme='light'] { /* light theme variables */ }

/* Key variables */
--font-size-md: 1.125rem;        /* Base UI font size */
--border-radius-md: 0.375rem;    /* Standard border radius */
--color-primary-600: #003041;    /* Primary brand color */
```

## Usage

```tsx
import { Button, TextInput, Dialog } from 'ui-mini';

function App() {
  return (
    <div>
      <TextInput label="Email" placeholder="Enter email..." />
      <Button>Submit</Button>
    </div>
  );
}
```
